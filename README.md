Requerimientos funcionales

1. El sistema debe permitir que el usuario escriba una pregunta o mensaje.
2. El sistema debe enviar la pregunta del usuario al sistema de inteligencia artificial.
3. El sistema debe recibir la respuesta generada por la inteligencia artificial.
4. El sistema debe mostrar la respuesta al usuario en la pantalla.
5. El sistema debe permitir que el usuario haga varias preguntas seguidas.

Requerimientos no funcionales

1. El sistema debe mostrar la respuesta al usuario en un tiempo máximo de 5 segundos.
2. El sistema debe poder ser usado desde un navegador web como Chrome o Firefox.
3. El sistema debe estar disponible para los usuarios al menos el 95% del tiempo.
4. La interfaz del sistema debe ser simple y fácil de entender para nuevos usuarios.
5. El sistema debe permitir que al menos 10 usuarios lo utilicen al mismo tiempo sin fallar.
